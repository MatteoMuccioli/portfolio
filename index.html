Sistema di Sicurezza a 4 Livelli

L1
Filtro Applicativo

Regex contro Prompt Injection, token speciali (es. <im_start>), Jailbreak. Limite lunghezza input.

L2
System Prompt Blindato

Istruzioni di sistema isolate su canale prioritario Gemini, impossibili da sovrascrivere dall'utente.

L3
Ancoraggio al Contesto

Temperatura 0.2 + frase di fallback obbligatoria quando nessun chunk è pertinente. Allucinazioni azzerate.

L4
Safety Filters Google

Filtri nativi Google Gemini per contenuti dannosi come layer finale di protezione.

Sicurezza di Contorno

Nonce WordPress + Cloudflare Turnstile (anti-CSRF/bot)
Rate limiting per IP
Token Bearer per comunicazioni interne (nascosto a Internet)
Retry con exponential backoff per errori 429/503
